# 그림으로 말해요 — 자폐 아동용 AAC 앱

그림 카드를 누르면 음성이 나오는 단일 상징 터치형 보완대체의사소통(AAC) 웹앱입니다.
안드로이드 태블릿 + Chrome 환경을 기준으로 만들었습니다.

## 주요 기능

- **단일 상징 터치 → 음성 출력**: 카드를 누르면 크게 보여주며 말해줍니다.
- **엄마/아빠 목소리 (보이스 뱅킹)**: 보호자 설정의 "목소리 일괄 녹음"으로 모든 단어를
  부모 목소리로 녹음해두면, 아이가 카드를 누를 때 그 목소리가 재생됩니다.
  녹음이 없는 단어는 한국어 TTS(기계 음성)가 대신 읽어줍니다.
- **그림 3종 지원**: 이모지(기본 제공) / 사진·파일 직접 등록(카메라·갤러리) /
  ARASAAC 오픈 라이선스 그림 상징 검색(CC BY-NC-SA, 비상업용).
- **보호자 잠금**: 톱니바퀴를 3초 길게 누른 뒤 곱셈 문제를 풀어야 설정·편집에 들어갑니다.
- **오프라인 동작**: 모든 데이터(사진·녹음 포함)는 태블릿의 IndexedDB에 저장됩니다.
  ARASAAC 검색만 인터넷이 필요합니다.
- **백업/복원**: 설정에서 JSON 파일로 내보내기/가져오기 (사진·녹음 포함).

## 태블릿에 설치하는 방법

1. 이 폴더를 HTTPS로 호스팅합니다. 가장 쉬운 방법은 GitHub Pages입니다
   (저장소 Settings → Pages → 브랜치 main / 루트 선택). 주소는 `https://<계정>.github.io/<저장소>/`가 됩니다.
   ※ 마이크 녹음과 PWA 설치는 HTTPS에서만 동작합니다.
2. 태블릿 Chrome으로 접속 → 메뉴(⋮) → **"홈 화면에 추가"** → 전체화면 앱으로 설치됩니다.
3. 처음 한 번 접속해두면 이후엔 인터넷 없이도 실행됩니다.
4. 아이가 다른 앱으로 나가지 못하게 하려면 안드로이드 **화면 고정**
   (설정 → 보안 → 앱 고정)을 함께 사용하세요.

## 처음 사용 순서 (보호자)

1. 톱니바퀴 3초 길게 누르기 → 곱셈 문제 풀기 → 보호자 설정 진입
2. "엄마 목소리 녹음하기"로 기본 단어 27개를 한 번에 녹음 (약 5분)
3. 목소리 선택에서 "엄마 목소리" 선택
4. 편집 모드에서 우리 아이에게 필요한 카드 추가
   (가족 사진, 좋아하는 간식 사진 등 실제 사진이 효과적입니다)
5. 백업 내보내기로 파일 저장해두기

## 기술 메모

- 빌드 없이 동작하는 바닐라 HTML/CSS/JS + PWA (Service Worker, IndexedDB)
- 음성: MediaRecorder(녹음) + Web Speech API(TTS 폴백)
